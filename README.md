# My First Laravel Steps 🚀

This is a simple project I built while learning Laravel basics.

<img width="1745" height="795" alt="my personal goals" src="https://github.com/user-attachments/assets/b64d7922-2ca1-49fa-98be-fbdc96fb7a10" />


### What's inside:
- **Routes**: Custom URL handling in `web.php`.
- **Views**: Clean templates using the **Blade** engine.
- **Logic**: Passing data arrays and using `@foreach` loops.

Everything is hand-coded to practice the fundamental "Route-to-View" flow.
